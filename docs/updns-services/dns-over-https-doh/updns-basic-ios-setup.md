# updns basic iOS setup

### 1. Install the [DNSCloak App](https://apps.apple.com/us/app/dnscloak-secure-dns-client/id1452162351) on your iPhone / iPad

### 2. Select `adfree.usableprivacy.net` as your default DoH server
